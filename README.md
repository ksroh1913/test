# KFTC 계좌 연동 대시보드 (qld.kr 벤치마크)

요청하신 내용대로 **qld.kr 대시보드 구조를 벤치마크**하고, **금융결제원 어카운트인포 API(계좌통합조회)**를 연동하는 5단계 개발 흐름으로 구성한 Next.js 프로젝트입니다.

## 벤치마크 핵심 포인트 (qld.kr)

- 상단에 핵심 KPI를 크게 노출(총 자산, 전일 변동)
- 자산 비중/랭킹 등 "한눈에 보는 요약" 중심
- 상세 내역(매매/입출금)은 아래로 배치
- 카드형 UI + 여백 중심 레이아웃

## 개발 단계 1~5

### 1) 벤치마크 & 요구사항 확정
- 목표: 개인 계좌 통합 잔액과 계좌 상태를 한 화면에서 확인
- KPI: 총 평가금액, 활동/비활동 계좌수, 상위 잔액 계좌

### 2) 환경 설치
```bash
npm install
npm run dev
```

### 3) 금융결제원 API 연동
- 서버 라우트: `app/api/accounts/route.ts`
- 실제 API 연동 로직: `lib/kftc.ts`
- 미설정 시 데모 데이터 반환(초기 개발용)

`.env.local` 예시:
```bash
KFTC_ACCESS_TOKEN=발급받은_access_token
KFTC_AUTH_CODE=사용자정보확인_후_받은_auth_code
```

### 4) UI 구현
- 페이지: `app/page.tsx`
- qld.kr 스타일 요약 카드 + 계좌 Top5 리스트

### 5) 배포
- Vercel 연결 후 환경변수 등록
- `KFTC_ACCESS_TOKEN`, `KFTC_AUTH_CODE`

## 주의사항

- API 토큰은 반드시 서버 환경변수로만 관리하세요.
- 계좌번호는 마스킹 후 화면 표시합니다.
- 실제 운영 전 사용자 동의/개인정보 처리 고지 절차를 반드시 정비하세요.
